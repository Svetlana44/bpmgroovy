# План тренировки LeetCode

## Цели

- 20–30 задач уровня Easy
- 5–10 задач уровня Medium (по одной на паттерн)

## Паттерны (Easy)

- массивы, строки
- hash map / set
- two pointers
- stack

## Рекомендуемый график

- Неделя 1: Arrays + Strings
- Неделя 2: HashMap + Two Pointers
- Неделя 3: Stack + повторение

## Контроль прогресса

- Каждая задача решена и тесты проходят
- В конце недели — 2 задачи на скорость
