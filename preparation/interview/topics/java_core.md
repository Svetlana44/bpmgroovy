# Java Core

## Вопросы и ответы

**1. Разница между checked и unchecked исключениями?**  
Checked должны быть обработаны или проброшены, unchecked — потомки RuntimeException.

**2. Что такое Generics и wildcard?**  
Generics задают тип параметра, wildcard (`? extends`, `? super`) ограничивают.

**3. Как работает equals/hashCode?**  
equals определяет логическое равенство, hashCode должен быть консистентен.

**4. Что такое immutability?**  
Неизменяемые объекты (String) удобны для безопасности и потоков.

**5. Что такое Optional и когда использовать?**  
Для явного выражения "может быть пусто", но не в полях DTO.

**6. Что такое аннотации и где используются?**  
Метаданные для runtime/compile‑time, например JUnit/TestNG.

**7. Что важно на Senior‑уровне?**  
Понимание JVM, памяти, производительности, корректного API дизайна.
