# Туториал 3: Collectors и группировка

## Частые коллекторы

- `toList`, `toSet`, `toMap` — базовые сборки.
- `groupingBy` — группировка по ключу.
- `partitioningBy` — разделение по предикату.
- `joining` — склейка строк.

## Примеры шаблонов

- Группировка пользователей по роли.
- Подсчёт частоты слов через `groupingBy` + `counting`.
- Построение `Map<K, V>` с обработкой дубликатов.

## Мини‑упражнения

1. Сгруппируй строки по длине.
2. Раздели числа на чётные/нечётные.
3. Построй `Map<String, Long>` с частотами.
