# Туториал 2: базовый Stream API

## Как строится pipeline

- Источник: коллекция, массив, `Stream.of`, `IntStream.range`.
- Промежуточные операции: `map`, `filter`, `flatMap`, `sorted`.
- Терминальные операции: `collect`, `reduce`, `count`, `forEach`.

## Важные свойства

- Ленивая обработка: выполняется только при терминальной операции.
- Стрим одноразовый: повторное использование вызывает ошибку.
- `map` меняет тип, `flatMap` разворачивает вложенные стримы.

## Мини‑упражнения

1. Преобразуй `List<String>` в `List<Integer>` по длине строк.
2. Из `List<List<Integer>>` получи плоский список.
3. Отфильтруй уникальные строки без изменения порядка.
