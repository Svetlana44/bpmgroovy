# Чеклист задач перед собеседованием (Java auto)

Формат: короткие задачи в стиле LeetCode, 30–60 минут в день.

## Коллекции / Map / Set

Easy:
- Two Sum
- Contains Duplicate
- Valid Anagram
- Intersection of Two Arrays
- Ransom Note
- First Unique Character in a String

Medium:
- Group Anagrams
- Top K Frequent Elements
- Longest Substring Without Repeating Characters

## Строки / массивы

Easy:
- Reverse String (StringBuilder.reverse())
- Valid Palindrome
- First Unique Character in a String
- Valid Anagram
- Longest Common Prefix
- Reverse Words in a String
- Move Zeroes
- Merge Sorted Array
- Missing Number

Medium:
- Reverse Words in a String III
- Group Anagrams
- Product of Array Except Self
- Subarray Sum Equals K

## Два указателя / окно

Easy:
- Remove Duplicates from Sorted Array
- Squares of a Sorted Array

Medium:
- Minimum Size Subarray Sum
- Longest Repeating Character Replacement

## Стек / очередь

Easy:
- Valid Parentheses
- Backspace String Compare

Medium:
- Daily Temperatures
- Evaluate Reverse Polish Notation

## Деревья (если спрашивают)

Easy:
- Invert Binary Tree
- Maximum Depth of Binary Tree

Medium:
- Binary Tree Level Order Traversal

## Мини‑JUnit практика

- ParameterizedTest на коллекции/строки.
- Assert‑сравнение массивов и списков.
- Подготовка тест‑данных и edge‑cases.
