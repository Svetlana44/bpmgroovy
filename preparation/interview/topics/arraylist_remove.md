# ArrayList: удаление элементов

## Как удаляются элементы

- `remove(int index)` — удаление по индексу
- `remove(Object o)` — удаление по значению

## Что происходит с размером

После удаления размер уменьшается на 1, элементы справа сдвигаются влево.
Сложность — O(n).

## Пример

```java
List<String> list = new ArrayList<>(List.of("a", "b", "c"));
list.remove("b"); // ["a", "c"], size = 2
```
