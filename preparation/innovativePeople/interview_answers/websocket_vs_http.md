# WebSocket vs HTTP

## Что такое WebSocket

Протокол для двустороннего (full‑duplex) обмена сообщениями по одному
TCP‑соединению.

## Отличие от HTTP

- HTTP — запрос‑ответ, соединение краткоживущее
- WebSocket — постоянное соединение и события в обе стороны

## Где применяют

Чаты, трейдинг, нотификации, онлайн‑игры, real‑time dashboards.
