# Примеры кода

Здесь собраны учебные примеры для практики. Их можно использовать как стартовые
шаблоны для собственных тестов.

## Разделы

- `contract/WiremockExampleTest.java` — примеры заглушек Wiremock
- `ui/PlaywrightExampleTest.java` — базовые UI тесты на Playwright
- `database/MongoDBExampleTest.java` — CRUD и запросы для MongoDB

## Как использовать

1. Откройте файл примера и посмотрите структуру теста.
2. Запустите локально (при необходимости поднимите сервисы).
3. Скопируйте шаблон и адаптируйте под свои сценарии.
