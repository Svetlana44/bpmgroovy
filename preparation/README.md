# Подготовка к собеседованию (Senior QA Automation Java)

Эта папка содержит материалы для подготовки к интервью, практику алгоритмов,
практические упражнения и аудит проекта с точки зрения best practices.

## Структура

- `interview/` — темы собеседования: вопросы + подробные ответы с примерами
- `practice/` — практические упражнения по MongoDB/Redis/Kafka
- `examples/` — примеры кода для практики (Wiremock, Playwright, MongoDB)
- `cheatsheets/` — шпаргалки (SQL и другие)
- `comparisons/` — сравнения технологий (например MongoDB vs Redis)
- `docs/` — план подготовки, советы, ресурсы
- `coding/leetcode/` — тренировка задач LeetCode‑типа с проверкой
- `coding/patterns/` — алгоритмические паттерны (задачи и шпаргалки)
- `project/` — улучшения проекта и сценарий демонстрации навыков

## Как использовать

1. Начать с `docs/PREPARATION_PLAN.md` и `interview/topics/README.md`.
2. Параллельно решать задачи в `coding/leetcode/`.
3. Пройти практику в `practice/` и свериться со шпаргалками в `cheatsheets/`.
4. Доработать проект по списку из `project/best_practices/`.
