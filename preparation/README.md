# Подготовка к собеседованию (Senior QA Automation Java)

Эта папка содержит материалы для подготовки к интервью, практику алгоритмов и
аудит проекта с точки зрения best practices.

## Структура

- `interview/` — темы собеседования: вопросы + подробные ответы с примерами
- `coding/leetcode/` — тренировка задач LeetCode‑типа с проверкой
- `coding/patterns/` — алгоритмические паттерны (задачи и шпаргалки)
- `project/` — улучшения проекта и сценарий демонстрации навыков

## Как использовать

1. Начать с `interview/topics/README.md` — список тем и порядок повторения.
2. Параллельно решать задачи в `coding/leetcode/`.
3. Доработать проект по списку из `project/best_practices/`.
